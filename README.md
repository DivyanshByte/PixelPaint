### Features
* Custom Pixel Size
* Custom File Extension (.ppa - Pixel Paint Art)
* Export in PNG
* Export with Transparency
* Many More!

### Keyboard Shortcuts
* Shift+Scroll Up or Down to Increase or Decrease Pen Size
* Alt+Scroll Up or Down to Increase or Decrease Eraser Size
